## Цель
Освоить маршрутизацию, Blade-шаблоны и организацию кода в Laravel 11 с использованием Tailwind CSS.

## Задачи и результаты

1) Создать контроллер `PageController`
- Команда:

```bash
php artisan make:controller PageController
```
- Файл: [my-personal-blog/app/Http/Controllers/PageController.php](my-personal-blog/app/Http/Controllers/PageController.php)
- Реализованы методы: `home()`, `about()`, `contact()`, `showPost($slug)` — возвращают соответствующие представления с параметром `title`.

2) Настроить маршруты
- Файл: [my-personal-blog/routes/web.php](my-personal-blog/routes/web.php)
- Добавлены маршруты:
  - `/` → `PageController@home` (имя `home`)
  - `/about` → `PageController@about` (имя `about`)
  - `/contact` → `PageController@contact` (имя `contact`)
  - `/post/{slug}` → `PageController@showPost` (с ограничением `where('slug', '[a-z0-9-]+')`)

3) Шаблоны Blade + Tailwind
- Базовый layout: [my-personal-blog/resources/views/layouts/app.blade.php](my-personal-blog/resources/views/layouts/app.blade.php)
  - Подключение ассетов через Vite: `resources/css/app.css`, `resources/js/app.tsx`.
  - Навигация, адаптивность, `x-cloak` правило и подключение Alpine.js CDN.
- Главная страница: [my-personal-blog/resources/views/pages/home.blade.php](my-personal-blog/resources/views/pages/home.blade.php)
- О нас: [my-personal-blog/resources/views/pages/about.blade.php](my-personal-blog/resources/views/pages/about.blade.php)
- Контакты: [my-personal-blog/resources/views/pages/contact.blade.php](my-personal-blog/resources/views/pages/contact.blade.php)
- Пост: [my-personal-blog/resources/views/pages/post.blade.php](my-personal-blog/resources/views/pages/post.blade.php)

4) Компонент формы
- Создан компонент через `php artisan make:component Form`.
- View компонента: [my-personal-blog/resources/views/components/form.blade.php](my-personal-blog/resources/views/components/form.blade.php)
- В `contact.blade.php` используется как `<x-form action="/contact" method="POST" buttonText="Отправить сообщение">`.

5) Адаптивная навигация и Alpine.js
- Навигация реализована с отображением ссылок на md+ и бургер-меню на мобильных.
- Alpine.js подключён через CDN в `layouts/app.blade.php`.
- Добавлено правило для `x-cloak` и fallback-класс `.mobile-only` с медиазапросом.

6) Валидация на стороне клиента
- В `contact.blade.php` добавлен скрипт, который проверяет наличие `@` в email и блокирует отправку с предупреждением.

7) Сборка ассетов
- Команда:

```bash
npm run dev
```

(в сессии разработки команда запускалась и Vite поднят для разработки)

## Как проверить локально

1. Установите зависимости (если нужно):

```bash
composer install
npm install
```

2. Запустите Vite (в отдельном терминале):

```bash
npm run dev
```

3. Запустите приложение:

```bash
php artisan serve --host=127.0.0.1 --port=8000
```

4. Откройте в браузере:
- Главная: http://127.0.0.1:8000/
- О нас: http://127.0.0.1:8000/about
- Контакты: http://127.0.0.1:8000/contact
- Пример поста: http://127.0.0.1:8000/post/hello-world
